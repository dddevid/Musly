## Installation

1. Install dependencies:
   ```bash
   flutter pub get
   ```
2. Run the app:
   ```bash
   flutter run
   ```

### Connecting to Your Server

1. Launch the app
2. Enter your Subsonic server URL (e.g., `https://your-server.com`)
3. Enter your username and password
4. Toggle "Legacy Authentication" if using an older server
5. Tap "Connect"
