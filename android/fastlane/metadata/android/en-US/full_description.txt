
Musly is a beautiful Flutter client for Subsonic-compatible music servers (Subsonic, Navidrome, Airsonic, Gonic). Stream music from your own server using an Apple Music-inspired UI that adapts to phones and tablets.

Features:
- Stream music from Subsonic-compatible servers
- Apple Music-style UI with automatic dark/light theming
- Powerful search, library browsing, and playlist management
- Full-featured Now Playing screen with shuffle, repeat, and queue controls
- Android Auto support and hardware media controls
- Offline playback and caching for listening without a connection
- Rich artist and album metadata with artwork

Getting started:
1. Launch the app and enter your Subsonic server URL, username, and password.
2. Toggle "Legacy Authentication" if your server requires it.

Musly is cross-platform (Android, iOS, Windows, Linux, macOS) and optimized for Android with a prebuilt APK available on the project's GitHub releases page.

Try the latest APK from the releases to test new features. Musly is open source and actively maintained on GitHub.
